# Taller 2 - Estructura de datos

## array unidimensional:
- ventasDiarias = [150, 200, 175, 220, 190]

- Cada posición representa las ventas (en unidades) de un día de la semana:

* Posición 0: lunes

* Posición 1: martes

* Posición 2: miércoles

*Posición 3: jueves

* Posición 4: viernes

---

## Matriz (2x3):

### Cada fila representa una categoría de productos:

- Fila 0: electrónicos (cantidades de 3 productos: laptops, tablets, audífonos)

- Fila 1: ropa (cantidades de 3 productos: camisas, pantalones, zapatos)

### Operación:
* Calcular el promedio de ventas por día a partir del array:
promedio = (150 + 200 + 175 + 220 + 190) / 5 = 187 unidades

Esto permite conocer el rendimiento diario promedio de la tienda.
